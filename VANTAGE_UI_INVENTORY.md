# Vantage Platform — Visual UI Inventory for Image Generation

## Executive Summary
Vantage is an AI-powered donor CRM for nonprofit organizations built with Next.js 16, Shadcn UI, and Lucide icons (1.5px stroke). The platform features a collapsible sidebar navigation, an AI chat overlay (Cmd+J), and a clean design system using OKLCH color tokens with light/dark mode support. Key visual identity: a green-to-cyan gradient (`#007A3F` to `#21E0D6`) reserved exclusively for AI-powered features.

## 1. Top-Level Pages & Routes

### Dashboard (`/dashboard`)
- **Purpose:** Primary entry point; overview of organization KPIs, AI insights, and recent activity
- **Key UI Elements:**
  - Metric cards (4-column grid: Total Revenue, Total Donors, Average Donation, YTD Revenue)
  - Daily Insights card (Sparkles icon, 1-4 insight items with colored icons: trending_up/emerald, heart/pink, alert/amber, star/blue)
  - Donations chart (bar chart showing giving trends)
  - Recent Gifts table
  - Top Donors widget
  - Onboarding Checklist (auto-hides when complete)
  - "Customize Dashboard" button

### Donor CRM (`/dashboard?view=donor-crm`)
- **Purpose:** Searchable data table of all organization donors
- **Key UI Elements:**
  - Advanced filters: search, date range, donor tags, assignee dropdown
  - Data table with sortable columns: name, email, phone, location, giving history, health score badge
  - Bulk actions: tag assignment, reassignment, merge, delete
  - Right-side panel: donor detail view with profile, interactions, notes, pledges, email template shortcuts
  - Donor health score: circular SVG ring with color-coded gradient (red→yellow→green, 0-100)
  - Merge dialog with visual confirmation

### Tasks (`/dashboard?view=tasks`)
- **Purpose:** Task management subview within Donor CRM
- **Key UI Elements:** Task list/table with status, assignee, due date, and donor link

### Donor Map (`/dashboard?view=donor-map`)
- **Purpose:** Mapbox-based geographic visualization of donor locations
- **Key UI Elements:**
  - Full-screen Mapbox GL map with custom drawing tools (circle, polygon)
  - Donor markers with popups (name, gift history, ID)
  - Sidebar with donor search, route optimization, and filtering
  - Export controls (CSV, report generation)
  - Zoom/pan controls, "Find My Location" button
  - Route report generation and download

### Donations (`/dashboard?view=donations` & `/dashboard/donations/entry`)
- **Purpose:** Donation history viewing and entry
- **Key UI Elements - Donations View:**
  - Filterable data table: donor name, amount, date, payment method, category, campaign, fund
  - Bulk acknowledgment controls (checkbox column)
  - Payment method badges (Check, Cash, Zelle, Wire, Venmo, DAF, QuickBooks)
  - Date range picker

- **Key UI Elements - Donation Entry:**
  - Form with fields: donor search (combobox), amount, date, payment method, category/campaign/fund dropdowns, memo
  - Submit button with validation

### Pipeline (`/dashboard/pipeline`)
- **Purpose:** Kanban-style sales pipeline for solicitation opportunities
- **Key UI Elements:**
  - 6-column Kanban board: To Contact, Qualified, Solicited, Committed, Closed Won, Closed Lost
  - Draggable opportunity cards (name, amount, date, donor link)
  - Column headers with count badges
  - "+ Add Opportunity" button per column
  - Card context menu: edit, delete, click to open donor profile
  - Inline opportunity edit dialog

### Saved Reports (`/dashboard?view=saved-reports`)
- **Purpose:** Custom report builder and saved report templates
- **Key UI Elements:**
  - Report templates grid (template cards with title, description, preview)
  - "Build Custom Report" button
  - Saved reports list/table with run date, creator, column picker

### Donor Profile (`/dashboard/donors/[id]`)
- **Purpose:** Deep-dive view of individual donor
- **Key UI Elements:**
  - Header: donor name, contact info (email, phone, address, LinkedIn/website links)
  - Tabs/sections: Overview, Interactions, Pledges, Insights
  - Donor Health Score card (circular SVG indicator)
  - AI Insights card (gradient border, "Generate Insights" button with Sparkles icon)
  - Magic Actions card (smart action buttons generated by AI)
  - Donor Notes card (rich text, editable)
  - Pledges card (pledge frequency, progress bar, next due date)
  - Tag badges on profile header
  - "Merge with Another Donor" button
  - Email template quick-send buttons

### Settings (`/settings`)
- **Purpose:** Organization and account configuration
- **Tabs/Sections:**
  - **Profile:** User name, email (read-only), role (read-only)
  - **Organization:** Org name, logo upload, year-end receipt settings
  - **Integrations:** QuickBooks (connect/sync button), Gmail (connect/disconnect), CSV import wizard
  - **Email Templates:** Template builder for donor acknowledgment and other emails
  - **Donation Options:** Category, campaign, and fund management
  - **Notifications:** Email notification preference toggles
  - **Team:** Team member invite/management
  - **Billing:** Stripe billing page embed, subscription status
  - **Audit Log:** Read-only log of org changes (exports available)
  - **Year-End Receipts:** Generate/send year-end tax receipts

### Auth Pages
- **Login/Signup:** Email/password forms
- **Join Org:** Invite acceptance page
- **Callback:** OAuth callback handler (hidden from users)

## 2. Feature Areas with Key Components

### AI Chat Agent (Vantage AI)
- **Purpose:** Conversational interface for donor queries, insights, and actions
- **Access:** Sidebar "Vantage AI" button or Cmd+J keyboard shortcut
- **UI Pattern:** Full-screen Sheet overlay (right-side slide-in) on desktop
- **Key Components:**
  - Chat header with "Vantage AI" title and close button
  - Message list area with:
    - User messages (right-aligned, solid background)
    - AI messages (left-aligned, markdown-lite rendering with donor links: `[Name](donor:id)` and bold: `**text**`)
    - Streaming text with `TextShimmerWave` animation
    - Tool execution badges (labeled cards showing: "Searched donors", "Looked up donor profile", "Calculated metrics", etc. with icons)
    - Custom card renderers: DonorCard, MetricsCard, TimeseriesChart, CompareCard, BuildCustomReportCard
  - Chat input bar at bottom with:
    - Text input field
    - Gradient send button (`from-[#007A3F] to-[#21E0D6]`, green→cyan)
    - Cmd+J hint text
  - Tool configuration icons (Search, UserCircle, BarChart3, Receipt, Activity)

### Dashboard Insights & Metrics
- **Purpose:** Quick-glance KPIs and AI-generated daily insights
- **UI Components:**
  - SectionCards: 4-column metric cards with gradient background (`from-primary/5 to-card`)
  - DailyInsights card: Sparkles icon in header, insight items with trending_up/heart/alert/star icons
  - ChartBarDonations: Responsive bar chart showing donation amounts over time
  - RecentGifts: Table of recent donations (donor, amount, date)
  - TopDonorsWidget: List of top donors by giving total

### Donor Health Score (AI-Generated)
- **Purpose:** Visual indicator of donor engagement and giving trends
- **Visual Design:**
  - Circular SVG progress ring (64px diameter by default)
  - Stroke color gradients: red (0-33) → yellow (34-66) → green (67-100)
  - Center text showing numeric score (0-100)
  - Trend icon (ArrowUpRight/green, ArrowDownRight/red, ArrowRight/blue) + trend label
  - Last updated timestamp

### Smart Actions (Magic Actions)
- **Purpose:** AI-suggested next steps for individual donors
- **UI Pattern:** Card with grid of action buttons
- **Each Action:**
  - Icon + action text (e.g., "Schedule follow-up call", "Send thank-you email")
  - Click to execute or open related dialog
  - Gradient accents on AI features

### Email Compose
- **Purpose:** Send personalized donor emails with template selection
- **UI Components:**
  - Dialog with header "Send Email to [Donor Name]"
  - Template selector (dropdown showing available email templates)
  - Recipient field (pre-populated with donor email, read-only)
  - Subject line input
  - Rich text editor for body
  - Preview pane
  - Send/Cancel buttons

### Donor Insights Panel
- **Purpose:** AI-generated analysis of donor patterns and recommendations
- **Visual Design:**
  - Gradient border card (`from-[#007A3F] to-[#21E0D6]` with 1px conic gradient border)
  - "AI Insights" title with Sparkles icon (teal color `#0ea5b8`)
  - "Generate Insights" button with gradient background
  - Loading state: Spinner + "Analyzing donor data..."
  - Results display: summary text + bulleted insights + next steps list

### Donation Acknowledgment System
- **Purpose:** Track and batch send donor thank-you communications
- **UI Components:**
  - Checkbox column in donations table
  - "Acknowledge Selected" button (bulk action)
  - Acknowledgment status badges (acknowledged/pending)
  - Template selection dropdown

## 3. Design System & Visual Details

### Color Tokens (OKLCH Format via CSS Variables)
**Light Mode:**
- Background: `oklch(1 0 0)` (white)
- Foreground: `oklch(0.145 0 0)` (near-black)
- Primary: `oklch(0.205 0 0)` (dark gray, used for buttons, text)
- Card: `oklch(1 0 0)` (white)
- Muted: `oklch(0.97 0 0)` (very light gray)
- Border: `oklch(0.922 0 0)` (light gray)

**Dark Mode:**
- Background: `oklch(0.145 0 0)` (near-black)
- Foreground: `oklch(0.985 0 0)` (near-white)
- Primary: `oklch(0.985 0 0)` (bright white text)
- Card: `oklch(0.145 0 0)` (near-black)
- Muted: `oklch(0.269 0 0)` (dark gray)

### AI Feature Gradient (EXCLUSIVE USE)
- **Hex Colors:** `#007A3F` (forest green) → `#21E0D6` (bright cyan)
- **Used For:** Chat buttons, AI feature headers, insight cards, magic action buttons, generate buttons
- **Pattern:** `bg-gradient-to-r from-[#007A3F] to-[#21E0D6]` with `text-white` and hover `opacity-90`
- **Animated Border:** Conic gradient rotating at 3s interval on insight cards

### Typography
- **Font Family:** Geist (sans-serif), Geist Mono (monospace)
- **Key Sizes:** Text-sm (12px), text-base (16px), text-lg (18px), text-xl (20px), text-2xl (24px), text-3xl (30px)
- **Font Weights:** normal (400), medium (500), semibold (600), bold (700)
- **Line Heights:** tight, snug, normal, relaxed

### Icons (Lucide)
- **Stroke Width:** 1.5px (consistent across all icons)
- **Common Icons Used:**
  - Navigation: LayoutDashboard, Users, Banknote, Target, FileText, Settings, Map, CheckSquare
  - Actions: Plus, Trash2, Edit, ExternalLink, Download, Copy, Mail, Phone
  - Status: CheckCircle2, AlertTriangle, Heart, Star, TrendingUp, ArrowUpRight, ArrowDownRight
  - Decorative: Sparkles (for AI), Building2 (org logo fallback)

### Shadows & Spacing
- **Shadow Classes:** `shadow-xs` (0 1px 2px), `shadow-sm` (standard)
- **Border Radius:** `rounded-lg` (0.625rem), `rounded-xl` (larger cards)
- **Spacing Scale:** px-2, px-4, px-6 (horizontal); py-4, py-6 (vertical); gap-2, gap-4, gap-6

### Component Patterns
- **Cards:** Shadcn Card with CardHeader, CardTitle, CardDescription, CardContent, CardFooter
- **Buttons:** Variant: "default" (primary), "outline", "ghost"
- **Modals:** Shadcn Dialog with DialogHeader, DialogTitle, DialogContent, DialogFooter
- **Dropdowns:** Shadcn DropdownMenu with trigger, content, items, separators
- **Tables:** Custom DataTable component with Tanstack Table (TanStack React Table)
- **Toasts:** Sonner toast notifications (success, error, info)
- **Form Elements:** Input, Label, Select, Checkbox, Switch, TextArea

## 6. Notable Interactions & Micro-UX

### Filters & Search
- **Date Range Picker:** Reusable component with preset ranges (Today, This Week, This Month, This Year, Custom)
- **Tag Filter:** Multi-select dropdown with tag pills
- **Assignee Filter:** Dropdown with "Unassigned" option
- **Search Box:** Real-time filtering as user types
- **Active Filter Badges:** Display selected filters as removable chips

### Bulk Actions
- **Selection:** Checkbox column in data tables
- **Actions Menu:** Appears when rows selected (assign tag, remove tag, reassign, merge, delete)
- **Confirmation Dialogs:** AlertDialog for destructive actions (delete, merge)

### Drag-and-Drop
- **Pipeline Kanban:** Drag opportunity cards between columns
- **Library:** dnd-kit (React drag-and-drop primitives)
- **Visual Feedback:** Card scales on drag, column highlights on hover

### Data Entry & Validation
- **Form Submission:** Sonner toast notifications for success/error
- **Loading States:** Spinner component, disabled buttons during async actions
- **Error Messages:** Inline field errors or toast error notifications

### Responsive Behavior
- **Sidebar:** Collapses to icon-only on mobile
- **Grids:** 1-column on mobile, 2-4 columns on desktop (via Tailwind `@container/main` queries)
- **Modals:** Full-screen on mobile, centered modal on desktop
- **Map:** Full-screen on all sizes, sidebar becomes collapsible drawer on mobile

## Summary for Image Generator

**Visual Style:** Clean, modern SaaS interface with minimal dark/light mode support. Heavy use of card layouts, tables, and grid systems. AI features visually distinguished by bright green-to-cyan gradient. Lucide icons at 1.5px stroke throughout. Generous whitespace and shadow-sm for depth. Sidebar navigation remains consistent across all main views. Chat overlay appears as full-screen or right-slide Sheet modal.

**Key Memorable Elements:**
1. Green-cyan gradient buttons and borders (exclusive to AI)
2. Circular donor health score SVG with color-coded ring
3. Kanban board for pipeline management
4. Mapbox geographic visualization
5. Right-side donor detail panel in tables
6. Sparkles icon consistently marking AI features
7. Sonner toast notifications (top-right corner)
8. Collapsible sidebar with icon-only mode

**Dominant Color Palette:** White/near-white background (light mode) or near-black (dark mode), with OKLCH-based grays. Green (#007A3F) and cyan (#21E0D6) as accent only for AI. Charts use OKLCH-defined color palette (chart-1 through chart-5).

**Typography & Spacing:** Geist font family, consistent 1.5px Lucide icons, generous gap-4/gap-6 spacing, rounded-lg/rounded-xl borders. Cards use shadow-xs, slightly lighter than standard shadows. Form inputs and buttons span consistent widths with label above pattern.
